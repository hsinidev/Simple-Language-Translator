A high-performance, responsive, and SEO-optimized language translation application. Built with modern web technologies, this tool provides instant text translation services wrapped in an immersive, cosmic-themed user interface.

## ğŸŒŸ Key Features

*   **Instant Translation**: Real-time text translation utilizing the LibreTranslate API.
*   **Immersive UI**: Features a custom-built, animated "Multi-colored Galaxy" starfield background using HTML5 Canvas.
*   **User-Centric Design**:
    *   Glassmorphism aesthetic for a modern, cleaner look.
    *   Dual-panel layout for easy source/target text comparison.
    *   One-click language swapping.
*   **SEO Optimized**: 
    *   Built-in Schema.org JSON-LD for rich search results.
    *   Comprehensive SEO content article integrated directly into the page.
    *   Optimized meta tags and open graph data.
*   **Compliance Ready**: Includes dedicated modals for Privacy Policy, Terms of Service, DMCA, and user guides.
*   **Fully Responsive**: Flawless experience across desktop, tablet, and mobile devices.

## ğŸ Getting Started

### Prerequisites

*   Node.js (v14 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/hsinidev/simple-language-translator.git
    cd simple-language-translator
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm start
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
